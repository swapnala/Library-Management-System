Welcome to Library-Management-System

This is a pet project created to provide a repository of the books available allowing users to browse through the listings and send a request to issue the book. The library SPOCS would approve the request and notify the user as to when to collect the book from the library. This project has been named as "Paperback" for Cummins Tech Centre.
